# Prerequisite

- Docker
- Makefile installed
- mysql-client should be installed

# How to start DB

```bash
make db_up
```

# How to destroy DB

```bash
make db_down
```

# How to migrate

```bash
make migrate
```

# How to test assignment

```bash
make test
```
